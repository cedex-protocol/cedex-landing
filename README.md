# CEDEX Contracts

Open-source repository for multi-chain NFT minting smart contracts supporting **EVM**, **Cedra**, and **Aptos** blockchains.

## What is this?

This repository contains smart contracts and frontend client for the CEDEX Genesis NFT collection - a role-based NFT system that allows users to mint on their preferred blockchain network.

## Supported Blockchains

- **EVM Chains**: Ethereum, Arbitrum, BNB Smart Chain (BSC), Polygon
- **Cedra**: High-performance Move-based blockchain
- **Aptos**: Layer 1 blockchain with Move VM

## Repository Structure

### [Smart Contracts](cedex-landing/contracts/)
Smart contract implementations for all supported chains. Includes tests, and configuration.

- EVM contracts (Solidity)
- Cedra contracts (Move)
- Aptos contracts (Move)

### [< Frontend Client](cedex-landing/client/)
Web application for interacting with the deployed contracts across multiple chains.
